# Portfolio — Arnaud BENACQUISTA

Portfolio personnel développé dans le cadre de ma reconversion professionnelle vers le développement logiciel.

---

## Aperçu

Site one-page au design glassmorphism "Deep Space Glass", inspiré de l'esthétique Apple visionOS.
Présentation de mon parcours, mes compétences, et mes projets réalisés en formation CDA à Metz Numeric School.

---

## Stack technique

| Élément | Choix |
|---|---|
| HTML | HTML5 sémantique |
| CSS | Tailwind CSS (CDN) + CSS custom |
| JS | Vanilla JS |
| Fonts | Syne + DM Sans (Google Fonts) |
| Icons | Font Awesome 6 |
| Logos | Devicons CDN |
| Formulaire | FormSubmit.co |

## Fonctionnalités

- Design glassmorphism avec orbes de fond animés
- Navigation sticky avec effet blur au scroll
- Menu burger responsive (mobile first)
- Filtrage des projets par catégorie
- Modals de détail pour chaque projet
- Animations au scroll via IntersectionObserver
- Cursor spotlight (desktop)
- Formulaire de contact via FormSubmit.co
- Accessibilité WCAG AA (focus visible, skip to content, prefers-reduced-motion)

---

## Lancer le projet en local

Aucune dépendance, aucun build tool nécessaire.

```bash
git clone https://github.com/ArnaudBena/PortfolioV2.git
cd PortfolioV2
# Ouvrir index.html dans un navigateur
```

Ou avec l'extension **Live Server** sur VS Code.
